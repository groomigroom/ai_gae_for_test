딥러닝 모델 훈련에서 사용하는 합성함수의 미분 원리

딥러닝의 훈련과 추론의 차이

딥러닝의 훈련(Training)은 방대한 데이터를 통해 모델이 규칙과 패턴을 스스로 학습하는 과정이며, 추론(Inference)은 학습이 끝난 모델에 새로운 데이터를 입력해 결과를 예측하고 판단하는 실전 활용 단계

경사기울기학습법 개념

경사기울기학습법(일반적으로 경사 하강법, Gradient Descent)은 머신러닝과 딥러닝에서 모델의 예측 오류(손실)를 최소화하기 위해 함수의 기울기를 활용하는 핵심 최적화 알고리즘입니다. 오류가 가장 적은 최적의 매개변수를 찾을 때까지 기울기의 반대 방향으로 반복해서 이동하는 원리입니다.

softmax 함수는?

출력값이 0~1 사이의 실수이며, 총합은 1인 함수

MSE, CEE 차이점

구분MSE (Mean Squared Error)CEE (Cross Entropy Error)
주요 목적연속형 데이터 예측 (예: 가격, 온도)범주형 데이터 분류 (예: 사진 속 동물 찾기)
핵심 개념예측값과 실제값 차이의 제곱에 대한 평균실제 확률 분포와 모델이 예측한 확률 분포 간의 차이 (엔트로피)
출력 형태실수형(연속된 값)확률값 (클래스별 0~1, 합은 1)
역전파 특징활성화 함수 미분값 포함으로 기울기 소실 문제 발생 가능성 상대적으로 높음활성화 함수 미분항이 상쇄되어 깔끔한 오차 역전파 가능

최적화 알고리즘 들 아래 3개까지
SGD

전체 데이터 대신 무작위로 추출한 단 하나의 데이터(혹은 일부)에 대해서만 기울기(Gradient)를 계산하여 가중치를 업데이트합니다.

Momentum (모멘텀)

: SGD에 관성(속도) 개념을 추가한 방식입니다. 현재 기울기 방향으로 이동하면서, 동시에 이전 단계에서 이동했던 방향과 크기를 일정 비율만큼 반영합니다.

Adagrad (에이다그라드)

변수들의 학습률(Learning Rate)을 개별적으로 다르게 조정하는 알고리즘입니다. 지금까지 많이 변한(업데이트된) 변수는 학습률을 작게 하고, 적게 변한 변수는 학습률을 크게 합니다.

Adam (아담)

Momentum(관성)과 Adagrad(학습률 자동 조정)의 장점을 결합한 최신 범용 최적화 알고리즘입니다. 기울기의 1차 모멘텀(평균)과 2차 모멘텀(분산)을 모두 고려합니다.

배치정규화 특징

배치 정규화(Batch Normalization)는 딥러닝에서 각 층의 입력 데이터 분포를 미니 배치 단위로 평균 0, 분산 1이 되도록 정규화하여 학습 과정을 안정화하고 속도를 크게 향상시키는 기법입니다.

CNN에서 Convolution 레이어의 특징

특징 맵(Feature Map) 생성. 지역적 공간 정보 보존. 가중치 공유 (Parameter Sharing). 이동 불변성 (Translation Invariance). 비선형성(Activation Function) 추가

CNN에서 활성화 맵(Activation Map)

특징 맵에 비선형 활성화 함수(예: ReLU)를 적용하여 뉴런의 활성화 여부를 최종적으로 확정한 출력 결과입니다.

CNN에서 특징 맵(Feature Map)

합성곱 층(Convolution Layer)에서 필터(Filter)가 입력 데이터를 순회하며 합성곱 연산을 통해 추출한 출력 데이터입니다.

Conv 레이어에서 활성화 맵의 출력 사이즈 계산
Conv 레이어에서 활성화 맵(Activation/Feature Map)의 출력 크기는 입력 크기, 필터(커널) 크기, 패딩(Padding), 그리고 스트라이드(Stride)를 사용하여 계산합니다.다음 공식을 통해 가로(Width)와 세로(Height) 크기를 각각 구할 수 있습니다.\(Output = \lfloor \frac{Input + 2 \times Padding - Kernel}{Stride} \rfloor + 1\)📝 변수 설명Input (입력 크기): 입력 데이터의 가로 또는 세로 크기Kernel (필터 크기): 합성곱 연산을 수행하는 필터의 크기Padding (패딩): 입력 데이터의 가장자리에 추가하는 0의 개수Stride (스트라이드): 필터를 이동시키는 간격\(\lfloor \dots \rfloor\): 소수점 이하를 버림(내림)하는 연산

pooling layer 중 AlexNet

오버래핑 풀링 (Overlapping Pooling) 특징: 기존의 겹치지 않는 풀링과 달리 윈도우 크기를 \(3 \times 3\), 스트라이드를 \(2\)로 설정하여 겹치는 영역을 허용. 이를 통해 과적합(Overfitting)을 방지.

pooling layer 중 VGGNet

2 * 2 Max Pooling 특징: 모든 풀링 레이어에 크기 \(2 \times 2\), 스트라이드 \(2\)인 Max Pooling을 사용하여 가로/세로 크기를 정확히 반으로 줄이는 규칙적인 체계를 도입

pooling layer 중 GoogLeNet

Inception 모듈 내부의 Max Pooling 및 Global Average Pooling (GAP) 특징:Inception 모듈 안에서 합성곱 연산과 함께 병렬적으로 3 * 3 Max Pooling을 수행.

pooling layer 중 ResNet

Convolution Stride를 통한 Downsampling 및 GAP 특징:풀링 레이어 자체의 사용을 최소화. 대신 특성 맵의 크기를 줄일 때 Pooling 계층 대신 Convolution의 Stride를 2로 설정하여 다운샘플링 수행.최종 출력층 직전에는 GoogLeNet과 유사하게 Global Average Pooling을 적용.

pooling layer 중 ReNet

: 기존 CNN의 핵심인 Convolution(합성곱) + Pooling(풀링) 구조를 RNN(순환 신경망)으로 대체한 모델입니다.

RNN

입력과 출력을 시퀀스(Sequence, 연속적인 데이터) 단위로 처리하는 딥러닝 모델입니다.

LSTM

순환신경망(RNN)의 일종으로, 시계열 및 자연어와 같은 순차 데이터의 패턴을 분석합니다.

Word2Vec에서 CBOW

주변 단어들을 통해 중심 단어를 예측하는 모델입니다.

Word2Vec에서 Skip-gram

중심 단어를 통해 주변 단어들을 예측하는 모델입니다.

GLove 모델 특징

단어 간의 의미적 관계를 벡터 공간에 표현(워드 임베딩)하는 대표적인 비지도 학습 알고리즘입니다.

Bert 모델 특징

트랜스포머의 인코더 구조를 활용한 자연어 처리(NLP) 모델

. , GPT2 모델 특징, One-Hot Encoding 특징 Attention 모델 특징, 매카니즘 을 간단히 파악 <강화학습> 마코프 결정 프로세스, 벨만 방정식 Q-Learning, e-greedy 알고리즘, DQN
